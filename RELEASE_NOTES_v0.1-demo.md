# 《往哲荣耀》（Philosophy Auto Chess）v0.1 Demo 版本说明

这是《往哲荣耀》的首个公开试玩 Demo。版本包含十波固定路线防守、25 名哲学家、四大阵营、六个小羁绊、8 人口成长、商店与三合一、法国革命节点、英国研究、启蒙议程、二阶柏拉图解锁的哲人王、W5“洞穴之影”、W10“绝对精神”、V6 稳定存档和局内平衡报告。

## 2026-07-16 修正版

- 正式战场地图已替换为最终美术，并归一到冻结的 1600×900 战斗坐标；路线、阻挡和部署位置未改。拖动棋子时底图会压暗，合法地面/高台位置分别显示金色/青色提示。
- 开波前会在地图入口短暂标出本波来敌路线和预计人数。
- 普通波次结束会播放一次仅覆盖战场地图的结算动画；购买经验有进度/升级反馈，刷新商店使用无文字的卡牌重排动画。
- 第五波“洞穴之影”现在是更明确的运营检查点：不升级、少人口且缺少体系的阵容可能阵亡或损失核心；第九、第十波强度保持不变。
- W2–W4 的敌压提升更明显：W2/W3 仍保留建队空间，长期只守两名一星棋子会从 W4 开始损失核心。W6 保留 Boss 后喘息，W7/W8 提前承接后期压力，使 W9 不再像突然跳变；W9/W10 数值未降低。
- 金币提示现在区分“预计/锁定利息”和“实际入账”，胜利结算会说明因 30 金币上限损失的溢出收入；经济公式与金币上限没有改变。
- 棋子阵亡只退出当前波战斗；波次结束后按原星级和原位置满血恢复，下一波可以继续上场。
- 局后报告继续记录每波阵亡，不会把阵亡棋子当作出售或永久损失。
- 修复成功结算与失败重试、战斗中读档之间的阵容恢复语义不一致。
- 修复非满血开波后战败重试会把哲人之石错误回满的问题；现在恢复本波开战检查点的生命值，不能通过失败重试治疗核心。

## 运行范围

- 支持 Windows 10/11 64 位。
- 压缩包自带运行时，无需安装 Node、npm 或其他开发环境。
- 完整解压后双击“启动往哲荣耀.cmd”即可使用默认浏览器游玩。
- 首次启动一定是新局；发行包不包含制作者的本地存档。

## 已知平衡观察项

以下项目属于试玩阶段的平衡观察，不作为程序 Bug：

1. 完美防守、利息与政治算术叠加后的后期经济宽裕程度。
2. W8—W10 对不同成型阵容的压力差异。
3. 德国六人体系的成型成本与后期收益。
4. 高治疗、高护盾阵容与纯输出阵容的通关容错差异。
5. 哲人王在前线少一人的代价与王城屏障收益是否平衡。

## 当前边界

- 地图美术仍为临时版本，地图坐标、道路和部署逻辑已经冻结。
- 本 Demo 不包含装备系统和新的角色或阵营。
- 正式模式隐藏校准层、测试台、作弊入口和详细开发日志；设置中保留只读的反馈报告导出。
- V6 存档迁移保留。战斗中退出后会安全返回该波准备阶段。
