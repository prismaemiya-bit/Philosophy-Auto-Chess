# 《往哲荣耀》v0.1 Demo 版本说明

这是《往哲荣耀》的首个公开试玩 Demo。版本包含十波固定路线防守、25 名哲学家、四大阵营、六个小羁绊、8 人口成长、商店与三合一、法国革命节点、英国研究、启蒙议程、二阶柏拉图解锁的哲人王、W5“洞穴之影”、W10“绝对精神”、V6 稳定存档和局内平衡报告。

## 运行范围

- 支持 Windows 10/11 64 位。
- 压缩包自带运行时，无需安装 Node、npm 或其他开发环境。
- 完整解压后双击“启动往哲荣耀.cmd”即可使用默认浏览器游玩。
- 首次启动一定是新局；发行包不包含制作者的本地存档。

## 已知平衡观察项

以下项目属于试玩阶段的平衡观察，不作为程序 Bug：

1. 完美防守、利息与政治算术叠加后的后期经济宽裕程度。
2. W8—W10 对不同成型阵容的压力差异。
3. 德国六人体系的成型成本与后期收益。
4. 高治疗、高护盾阵容与纯输出阵容的通关容错差异。
5. 哲人王在前线少一人的代价与王城屏障收益是否平衡。

## 当前边界

- 地图美术仍为临时版本，地图坐标、道路和部署逻辑已经冻结。
- 本 Demo 不包含装备系统和新的角色或阵营。
- 正式模式隐藏校准层、测试台、作弊入口和详细开发日志；设置中保留只读的反馈报告导出。
- V6 存档迁移保留。战斗中退出后会安全返回该波准备阶段。
