# Audio asset contract

The files in this directory are original procedural PCM assets generated by
`scripts/generate-audio-assets.mjs`. They contain no third-party samples and no
generative-model output.

Design targets:

- 44.1 kHz, mono, signed 16-bit WAV for universal browser decoding.
- Short tails, soft attacks and conservative peaks to avoid clicks and fatigue.
- Combat texture is deliberately quieter than decisions, warnings and results.
- Runtime gain, priority, cooldown and deterministic pitch variation remain
  centralized in `app/game/audio.ts`.

Run `npm run generate:audio` to reproduce the WAV files and
`TECHNICAL_REPORT.txt`.
