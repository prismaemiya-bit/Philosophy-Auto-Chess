# 《往哲荣耀》冻结地图美术规格

- 布局版本：`frozen-1600x900-v1`
- 本地画布：1600 × 900 px，16:9。所有道路、格位、单位、敌人、节点和特效共用此坐标面；运行时只能整张等比缩放，禁止 `cover` 裁切或图层分别缩放。
- 冻结指纹：`1b493ce5713f709ce439cf730149d89ae492f1cb9f09f0035af949764eb3802e`。除明确 Bug 修复外，不改变路点、20 个部署位、三个法国节点、入口、A/B 汇合、终点或核心坐标。

## 尺寸

- A/B 道路有效表面宽度：49 px；描边 58 px；阴影占宽 68 px。
- C 道路有效表面宽度：43 px；描边 52 px；阴影占宽 62 px。
- 普通棋子视觉框：82 × 102 px。
- 最大棋子视觉安全框：93 × 116 px。立绘、星级、生命/能量条和必要状态必须包含在此框内。
- Boss 占地：141 × 141 px；中心始终锚定当前道路中心点。
- 哲人之石中心及三路终点：`(1504, 450)`；核心战斗安全区：`x=1240..1584, y=250..650`。

## 锚点

- 部署数据坐标就是棋子脚底锚点。普通/最大棋子框的水平锚点为 50%，垂直锚点为框高的 54%；即框向上延伸 54%，向下延伸 46%。
- 地面棋子脚底锚点位于道路中心线；高台棋子使用同一规则但位于高台中心。
- 敌人与 Boss 使用中心锚点；Boss 占地圆随固定路点插值移动，不改变路线或阻挡判定。

## 前景遮挡层

- 推荐层级：纯净底图/远景 → 道路与平台 → 后景装饰 → 部署基础 → 单位与敌人 → 技能/漏怪反馈 → 前景遮挡。
- 前景遮挡不得覆盖道路中心线、部署锚点、最大棋子安全框、法国节点或核心战斗安全区；确需掠过时应使用独立透明前景层，并保证关键轮廓区域低于 20% 不透明度。
- 顶部 72 px 为提示安全区，不放入口名称、关键地标或不可替代的美术信息。
- 最终美术应按 1600 × 900 原尺寸交付；透明校验层必须能以 `(0,0)` 无偏移叠加验证。

施工包由 `npm.cmd run export:map-art` 直接读取 `app/game/positions.ts` 生成，不允许人工重描坐标。
