# 往哲荣耀 / Philosophy Auto Chess v0.2

v0.2 将历史事件、意识形态、战争机器、历史档案、任务记录、结算战绩、基础音频接口和手机横屏网页整合到同一套确定性游戏核心。

## 本版新增

- W3 历史事件与 W6 意识形态选择，结果进入 V7 稳定存档和波次检查点。
- 战争机器、多个具有独立阶段特征的 Boss，以及更平滑的 W5/W9 难度检查点。
- 历史档案、幂等任务记录和可分享的阵容、羁绊、伤害、承伤、治疗/护盾结算。
- 桌面与手机浏览器共用购买、部署、出售、战斗、历史事件和存档逻辑。
- 手机横屏触控操作、地图底缘备战棋子带、独立五卡商店和安全区适配。
- 玩家可用的 JSON 存档导出/导入；导入失败不覆盖现有进度，成功导入前自动创建恢复备份。
- PWA Manifest、主屏幕图标、standalone 元数据和保守的在线优先更新策略。
- 音乐与音效分轨接口、音量和静音设置；资源缺失不会阻断游戏。

## 兼容性

- 当前局内存档版本为 V7，继续兼容 V1—V6 合法存档。
- 局外档案版本为 V3，与局内存档独立。
- 未来版本存档会被拒绝并保留原文件，不会被当前版本降级覆盖。
- 战斗中的临时敌人、计时器和事件队列不会写入稳定存档；重新进入时从本波准备阶段恢复。

## 已知边界

- 存档默认保存在浏览器本地；不提供账号或云存档，清理浏览器数据可能丢失进度。
- PWA 当前提供可安装元数据与离线说明页，不承诺完整离线游玩。
- 仓库没有最终音乐或录制音效文件；当前使用可静音的低音量合成回退。
- Android Chrome 与 iPhone Safari 的最终真机验收仍需在正式 HTTPS 地址上完成。
- 本版不包含装备、多人、登录、支付、广告、统计追踪、新地图或局外数值成长。
