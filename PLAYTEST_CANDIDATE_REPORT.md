# 《往哲荣耀》V0.2 本地试玩候选报告

日期：2026-07-23  
性质：本地候选，不是已发布版本  
正式版本元数据：仍为 0.1.1

## 建议优先体验的主流候选

1. **快速升级均衡混编**
   - 前期只买能立即上场或接近二星的棋子，第二波起优先购买经验。
   - W5 前达到约 6 人，W8 前争取 7—8 人；不要为了满利息一直停在 3—4 人口。
   - 96 局经营样本中，快速升级 W8 为 24/24、W10 为 7/24，是唯一稳定到达后期的 AI 策略。

2. **高阻挡前排 + 有效远程输出**
   - 费希特、霍布斯等承担三路接敌，后排使用群攻、单体和控制，不要只堆坦克。
   - 固定阵容的高阻挡前排在三种遭遇种子的 W8—W10 为 9/9；普通前排包只有 6/9，说明容错确实存在，但输出仍不可缺。

3. **哲人王体系**
   - 费希特王与笛卡尔王样本均为 W8—W10 9/9，且阵容价值约 45，属于当前效率较高的稳定候选。
   - 请重点观察王座是否让普通部署显得没有选择价值，以及王城屏障是否在真人操作中太容易覆盖三路。

4. **法国混搭契约或启蒙**
   - 法国+契约、法国+启蒙固定板均为 9/9，比只有六名法国成员的 6/9 更稳定。
   - 当前更像“法国空间控制作为核心，再用前排/小羁绊补三路”，不建议为了纯阵营强行放弃功能位。

5. **英国证据与逻辑分析**
   - 完整英国固定板为 8/9，经营矩阵 W8 17/32，是三个原型中前中期最好的一组。
   - 同板将罗素替换为亚里士多德达到 9/9；请观察罗素在随机 Boss 中是关键解法，还是只有特定目标才有价值。

## 当前偏弱或更挑条件的玩法

- **只存利息**：W8 3/24、W10 0/24；大量收入在 30 金币上限处溢出，人口和战力跟不上。
- **前期连续盲刷**：W8 6/24、W10 0/24；二星形成速度没有抵消人口落后。
- **纯续航阵**：固定样本 W8—W10 为 6/9、W10 0/3；能拖延但缺少结束 Boss 的单体输出。
- **希腊纯控制**：6/9、W10 0/3；W9/W10 会暴露输出与前排耐久不足。希腊控制在混编里仍可能强，暂不等同于希腊角色整体弱。
- **纯法国六人**：6/9、W10 0/3；加入契约、启蒙或更可靠前排后显著改善，重点检查法国是否过度依赖外援。
- **只堆普通前排**：6/9、W10 0/3；三路覆盖与击杀速度不足。
- **无体系低费乱配**：4/9；符合游戏应惩罚缺乏阵型和升级的设计目标。

## 历史事件重点对照

- 强势观察：五月风暴 + 激进主义，AI 矩阵 8/27，为29种合法组合最高。
- 次强观察：五月风暴 + 改良主义/自由主义/共产主义，各 6/27。
- 弱势观察：宗教改革 + 改良主义、世界大战 + 自由主义、世界大战 + 共产主义，均为 0/27。
- 这些不是正式真人胜率。请特别记录事件是否改变了本来会做的运营决定，而不是只看最终通关。

## 建议你的试玩顺序

1. 快速升级均衡混编，记录每波等级、人口和剩余金币。
2. 同样运营一次只存利息，比较 W5、W8 的阵容规模。
3. 费希特王或笛卡尔王，观察屏障和王座是否过强。
4. 纯法国六人，再玩法国+契约或启蒙，比较 W9 三路覆盖。
5. 希腊控制或纯续航，观察 W10 是时间不足、前排死亡还是 Boss 漏过。
6. 五月风暴+激进主义，以及世界大战+自由主义/共产主义各一局。
7. 在设置中分别调整音乐、音效音量，刷新页面确认分轨设置保留；当前未注册正式音乐文件时音乐保持安静属于预期。

每局建议记录：最终波次、W8/W9核心生命、W10 Boss种类、阵亡人数、失败原因、你认为的关键棋子和是否出现“明显只有一种正确选择”。

本候选没有新增角色或地图，没有修改基础经济与战斗数值，也没有打包、发布或上传 GitHub。
