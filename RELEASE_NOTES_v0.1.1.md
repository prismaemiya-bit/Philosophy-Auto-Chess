# 《往哲荣耀》（Philosophy Auto Chess）v0.1.1 版本说明

开发：折射棱镜

本次补丁是第一轮代码与平衡审计后的稳定 Windows 便携版本，继续兼容 V6 存档。

## 关键变化

- 防御属性正式进入战斗计算，敌人攻击会读取目标防御并按统一规则减伤。
- 高阻挡前排会承担更多并发敌人的接触伤害，阻挡强度不再只有收益而没有承压代价。
- 商店卡牌与单位检查器显示防御值，玩家可以在购买和布阵前比较生存能力。
- 棋子阵亡仅影响当前波次，结算后按原星级和位置恢复；失败重试恢复本波开战检查点。
- 平滑前中后期波次压力，并增加确定性战斗、连续重试、存档边界、局后报告和地图拖放回归测试。

## 验证

- TypeScript、eslint 与 production build 通过。
- 自动测试 90/90 通过。
- Windows 便携包在含中文与空格的解压路径中完成全新开局、购买、部署、第一波、关闭重启和 V6 续档测试。

完整解压 ZIP 后，双击 `启动往哲荣耀.cmd` 或 `start-game.cmd`。压缩包自带运行时，不需要安装 Node，也不需要联网。
