# 《往哲荣耀》V0.2 历史事件确定性 AI 审计

## 审计边界

- 运行入口：`npm.cmd run history:sim`，实现位于 `scripts/historical-event-sim.ts`。
- 样本：7 个历史事件的 29 个合法“事件 × 意识形态”组合，固定种子 101 / 307 / 503，存息 / 前期刷新 / 快速升级 3 种策略，常规 / 弱势分散 / 法国专精 3 种阵容代理，共 783 局。
- 每次执行会在内存中完整运行矩阵两遍并逐字节比较结果；最终结果 `deterministicReplay: true`。
- W3 与 W6 只强制指定待出现的事件和合法候选，过门禁仍调用生产动作 `confirmNormalEvent`、`chooseReformationReward`、`chooseRealityStance`。工业免费刷新、改良替换、自由完整退款也调用正式引擎动作。
- AI 是固定启发式，不等于真人胜率。尤其“弱势分散”代理在低费追牌中较易合成，最终通关率反而略高；这说明该标签不能用来推断角色强弱，只能用于覆盖异质牌序。

### 意识形态与一次性动作

- 改良主义替换实际执行 574 次，自由主义完整退款执行 503 次；逐波 flag 均正常重置，没有重复使用或商店门禁。
- 激进主义总体通关率 17.28%，但只会出现在城邦危机、法国大革命、五月风暴三个可强化事件中，不能和五事件覆盖的其他意识形态直接横比。
- 同事件对照中：城邦危机激进主义 7.41%，法国大革命 14.81%，均未超过同事件最佳档；五月风暴激进主义为 29.63%，高于其余持续事件档 22.22%，但 27 局样本尚不足以证明失控。
- 共产主义总体 9.88%，自由主义 8.99%，改良主义 8.47%，没有观察到公共供给、完整退款或替换形成必胜循环。
- 保守主义在五月风暴会使部署上限回落。模拟器通过正式出售动作处理满备战区溢出后，死锁从 2 降为 0；这不是引擎卡死，但教学应提醒玩家可能需要撤回或出售一枚棋子。

## 世界大战反事实与实际修改

初始规则让战争机器直接乘基础波次生命倍率：W4 1550、W7 3224、W9 4340 且 W9 同时两台；移动时再承受 50% 伤害。首轮矩阵中 W7/W9 击破数和奖励数均为 0，奖励机制在后两次出现中事实上不可达。

本轮只给战争机器增加事件波次生命倍率上限：W4 2.5（保持 1550）、W7 3（1860）、W9 2（1240）。不改变基础敌群、波次倍率、机器数量、两路线、移动减伤、召唤、奖励、经济或角色数值。

| 波次 | 修改前击破/领奖 | 修改后击破/领奖 | 判断 |
|---:|---:|---:|---|
| W4 | 88 / 88 | 88 / 88 | 完全不变 |
| W7 | 0 / 0 | 15 / 15 | 奖励恢复为可达，仍有明显附加压力 |
| W9 | 0 / 0 | 4 / 1 | 双路线全击破仍稀有，但不再逻辑上不可达 |

世界大战通关数仍为 3/108，说明修正没有把高压事件改成送分项。W9 的平均伤害不能直接做前后因果比较，因为 W7 领奖会改变后续购买状态和到达样本；本轮以“击破与原子领奖从 0 恢复”为最小修复标准，不继续削弱基础 W9。

## 延期与真人验收

- 暂不修改基础收入、利息、金币上限、刷新/经验价格、商店概率、人口价格或角色面板。
- 暂不削弱五月风暴/激进主义；29.63% 是值得真人观察的局部优势，不是 27 局固定启发式即可判定的 P1。
- 真人连续局重点：W6 选择保守主义后的撤回提示、W7/W9 战争机器是否能看懂“移动减伤—阻挡解除减伤—击破领奖”、W8/W9 失败归因是否清楚。
